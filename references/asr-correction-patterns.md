# ASR常见错误修正模式

中文语音识别(ASR)的典型错误类型和修正思路。适用于演讲、教学类内容。

## 1. 同音/近音替换

最常见的ASR错误。需要根据上下文推断正确用字。

| ASR输出 | 正确 | 上下文线索 |
|---------|------|-----------|
| 严实出高徒 | 严师出高徒 | 成语"严师出高徒" |
| 盔子生意 | 亏本生意 | 常见口语表达 |
| 成小培养 | 从小培养 | "从小培养小孩" |
| 运心去领悟 | 用心去领悟 | "用心"是常用词 |
| 会命 | 慧命 | 佛学/教育术语 |
| 引习 | 尹喜 | 历史人物（关令尹喜） |
| 进菩萨 | 近菩萨 | "近"表示靠近 |
| 衣贫如洗 | 一贫如洗 | 成语 |
| 显不出 | 写不出 | 上下文讲写诗 |
| 大气蓬勃 | 大气磅礴 | 成语"大气磅礴" |
| 简单情细 | 剑胆琴心 | 成语"剑胆琴心" |

## 2. 成语/俗语误识别

ASR经常把不熟悉的成语拆成同音字组合。

| ASR输出 | 正确 | 备注 |
|---------|------|------|
| 属无父级磁力 | 手无缚鸡之力 | 12字→7字，语音连读 |
| 人生下巴厂 | 人生下半场 | "下巴厂"→"下半场" |
| 取法乎上得其尊严 | 取法乎上得其中 | 古文引用需查原文 |
| 取法乎尊得其下严 | 取法乎中得其下 | 同上 |

**处理策略**：对4字以上疑似成语/俗语的片段，搜索验证后再改。

## 3. 专有名词误识别

| ASR输出 | 正确 | 类型 |
|---------|------|------|
| 公沙角子语 | 宫商角徵羽 | 五音（音乐术语） |
| 礼乐是预书书 | 礼乐射御书数 | 六艺（教育术语） |
| 相述理占 | 相术理占 | 术数术语 |
| 皮尤朵朵 | 老婆孩子 | 需听音频确认 |
| 孔老夫子周游内国 | 孔老夫子周游列国 | 历史典故 |
| 真国方 | 曾国藩 | 人名 |

**处理策略**：涉及古文、历史、音乐等专业领域时，查证后再改。

## 4. 通假/方言音变

某些方言区说话者，ASR会产出特定的音变模式。

| ASR输出 | 正确 | 规律 |
|---------|------|------|
| 运心 | 用心 | y/n 声母混淆 |
| 不也给他 | 也不给他 | 词序错误 |
| 搞死你搞不明白 | 搞到死也搞不明白 | 口语省略 |
| 都令营了 | 都做了 | "了"被扩大 |
| 做不好小孩子办的 | 做不好小孩子教育的 | "办"→"教育" |

## 5. "音/因"通假（特殊场景）

涉及佛教/因果主题的内容，ASR常把"因"识别为"音"。

| ASR输出 | 正确 |
|---------|------|
| 花代表音 | 花代表因 |
| 种音得善果 | 种因得善果 |
| 音果 | 因果 |
| 音上努力 | 因上努力 |
| 花就是音 | 花就是因 |

**注意**：必须确认上下文确实讲的是"因果"主题才批量替换。"音乐"的"音"不能改。

## 6. 不应修改的情况

以下情况即使看起来奇怪也**不要改**：
- 口语化表达："你放心好了"、"效果不一样"、"一大堆"
- 语气词重复："啊我终于可以做我自己喜欢做的事情"
- 口语倒装："但是是你应该做的"（虽然"但是是你"看起来奇怪，但口语中确实这样说）
- 方言特色表达（除非确认是ASR错误）

## 7. 修正词典构建流程

1. **通读全文**，理解内容主题和说话风格
2. **标出疑似错误**，区分"确定错"和"可能错"
3. **查证**成语、古文、专有名词
4. **构建词典**：只包含确定的修正
5. **标记存疑**：不确定的用 `[?]` 标记，交给用户确认
6. **运行脚本**，查看变更明细
7. **抽查验证**：重点看拆分结果和修正是否合理
